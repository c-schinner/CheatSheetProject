Beginner Cheatsheet for HTML/CSS

  • Tags
  • Selectors
  • Colors
  • Images
  • Properties

Tags

Tag Name Description
<table> Table the wrapper elemment for all HTML tables
<thead> Table Header the set of rows defining the column headers in a table
<tbody> Table Body the set of rows containing actual data
<tr> Table Row the table row container
<td> Table Data the data going into the row
<tfoot> Table Footer the set of rows defining the footer of a table

Selectors

Selector Description
p Paragraph, or any other text
h1 - h6 Heading 1 through Heading 6
ul and ol Unordered and Ordered lists
a Anchor for links
classes Classes use the .class-name syntax
id ID's use the #id-name syntax

Colors

Color name Description
red Red color
blue Blue color
green Green color

Images

Syntax Description
src: Source url for the image
alt: Alternative Description (for screen readers)
width: Width of the image

Properties

Syntax Description
font-family changes the style of the font
font-size changes the size of the font
font-weight changes the weight of the font (boldness)
text-align changes the alignment of the text (left, right, center)
background-image adds an image as the background of the element url(add the url here)
background-color changes the color of the background
border adds a border to the element
border-radius changes the radius of the border
padding adds padding to the element(top, right, bottom, left) (indside the element)
margin adds margin to the element(top, right, bottom, left) (outside the element)
color changes the color of the text
line-height changes the height of the line of text
display changes the display of the element

There are many more tags, selectors, colors and properties. And there is a lot more information to learn.
If you want to learn more about HTML/CSS, check out the MDN CSS documentation.
